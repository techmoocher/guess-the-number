# Guess the Number

This is a simple number guessing game where the player has to guess 
a randomly generated number from 1 to 100 within a limited number 
of attempts. This project is originally designed to be a submission 
for the [Roadmap.sh](https://roadmap.sh) project.

Find out more at 
[https://roadmap.sh/projects/number-guessing-game](https://roadmap.sh/projects/number-guessing-game)
